# Login Help

## Password reset

Use **Forgot password** on the sign-in page. Reset links expire after 60 minutes and can only be used once.

## Two-factor authentication

Enable 2FA under **Settings → Security**. Backup codes are shown once—store them securely. Lost device access requires identity verification via support.

## Account lockout

Five failed sign-in attempts lock the account for 15 minutes. SSO users should sign in through their organization provider instead of email/password.
